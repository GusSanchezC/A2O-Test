Problem 2: String value

Jane loves strings more than anything.
She has a string t with her, and value of string s over function f can be calculated as given below:

f(s) = |s| x Number of times s occurs in t

Jane wants to know the maximum value of f(s) among all the substrings (s) of string t. Can you help her?
The input is a single line containing string t.
Print the maximum value of f(s) among all the substrings (s) of string t.

Sample input 1

aaaaaa

You can see the explanation and the result of this input in the Explanation 1 on the right side.

Sample input 2

abcabcddd

You can see the explanation and the result of this input in the Explanation 2 on the right side.

Explanation 1: aaaaaa

f('a') = 6
f('aa') = 10
f('aaa') = 12
f('aaaa') = 12
f('aaaaa') = 10
f('aaaaaa') = 6

In this case, the maximum value of f(s) among all the substrings (s) of string t its 12.

Explanation 2: abcabcddd

f("a") = 2
f("b") = 2
f("c") = 2
f("ab") = 4
f("bc") = 4
f("ddd") = 3
f("abc") = 6
f("abcabcddd") = 9

In this case, the maximum value of f(s) among all the substrings (s) of string t its 9.
